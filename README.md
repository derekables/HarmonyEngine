# Harmony Engine

A browser-native music composition platform inspired by the workflow efficiency of classic tablature editors, rebuilt as a modern extensible music engine.

## Vision
Harmony Engine is not just a tablature editor. It is a unified music system where tablature, notation, MIDI, playback, AI assistance, and education tools all operate on a single canonical music model.

## Core Principles
- Speed above everything
- Keyboard-first workflow
- Musical truth over visual representation
- Progressive complexity
- Non-destructive editing
- Platform independence
- AI as assistant, not author

## Status
Early foundation phase. Architecture and core engine design in progress.
